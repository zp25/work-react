# work-react

react起始模版

webpack 2 + redux + react router 4 + react hot loader 3

## 目录
app目录

~~~
+-- actions
+-- assets
+-- components
|   +-- app.jsx
+-- constants
|   +-- actionTypes.js
+-- containers
|   +-- root
|   |   +-- dev.jsx
|   |   +-- index.jsx
|   |   +-- prod.jsx
|   +-- app.jsx
|   +-- devtools.jsx
+-- images
+-- lib
|   +-- createAction.jsx
|   +-- createReducer.jsx
|   +-- index.jsx
+-- reducers
+-- store
+-- styles
+-- index.jsx
~~~
assets目录存放其他资源，使用copy-webpack-plugin复制到dist目录
